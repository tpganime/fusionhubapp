FUSIONHUB ANDROID APP SETUP GUIDE

1. Open Android Studio and create a new project.
   - Template: "Empty Views Activity"
   - Name: FusionHub
   - Language: Java
   - Minimum SDK: API 24 (Android 7.0)

2. Copy Code:
   - Copy contents of `android_export/AndroidManifest.xml` -> `app/src/main/AndroidManifest.xml`
   - Copy contents of `android_export/activity_main.xml` -> `app/src/main/res/layout/activity_main.xml`
   - Copy contents of `android_export/MainActivity.java` -> `app/src/main/java/com/[your-package]/MainActivity.java`

3. Configure:
   - In MainActivity.java, change the package name at the top to match your project.
   - In MainActivity.java, change the APP_URL string to your deployed website URL (e.g. https://your-app.vercel.app).

4. Build & Run:
   - Press the Play button in Android Studio to run on your emulator or connected device.

Note: Ensure your web app is deployed (Vercel/Netlify) before running the Android app, as it loads the live URL.
